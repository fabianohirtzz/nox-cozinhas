# NOX Cozinhas Industriais — Site Institucional

## Estrutura do Projeto

```
nox-project/
├── index.html                  ← Página principal
├── assets/
│   ├── css/
│   │   └── style.css           ← Todos os estilos
│   ├── js/
│   │   └── main.js             ← Scripts (cursor, loader, produtos, setores...)
│   ├── images/
│   │   └── logo-nox.png        ← Logo oficial da marca
│   └── video/
│       └── INSTRUCOES.txt      ← Instruções para adicionar o vídeo hero 9:16
```

## Como usar

Abrir `index.html` diretamente no navegador (Chrome, Edge, Firefox).
Não requer servidor, build ou dependências externas.

## Vídeo Hero

Adicionar `hero-video.mp4` (formato 9:16) em `assets/video/`.
Ver `assets/video/INSTRUCOES.txt` para especificações completas.

## Fontes

Carregadas via Google Fonts (requer internet):
- Bebas Neue (títulos)
- Barlow (corpo)
- JetBrains Mono (dados técnicos)

## Cores Oficiais

- Amarelo marca: `#ffc500`
- Fundo principal: `#070707`
- Texto:          `#ece8e0`

## Seções
